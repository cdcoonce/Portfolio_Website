# Add Oura Pipeline Project to Portfolio

## Context

Add the [Oura-Pipeline](https://github.com/cdcoonce/Oura-Pipeline) project — a personal ELT pipeline that pulls Oura Ring health data via the Oura API v2, loads it into **Snowflake** (VARIANT columns for semi-structured JSON), and transforms it through **dbt** staging/mart models into analysis-ready tables, all orchestrated by **Dagster** with daily partitions. Uses RSA key-pair auth for Snowflake and OAuth2 for the Oura API (tokens stored in Snowflake). The Snowflake migration is on `feat/snowflake-migration` branch. This will be a **featured** project, replacing Manufacturing Downtime Analysis in the featured set (keeping the total at 4).

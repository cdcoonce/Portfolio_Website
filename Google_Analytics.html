    Google Analytics

    In my role as a Business Analyst, I spearheaded a significant Google Analytics 3 (GA3) data migration for nearly 70 properties. Starting with stakeholder requirement gathering and approval of project drafts, I then utilized Python and the GA3 API to extract comprehensive datasets and create a data dictionary to support data usability.

    I developed a Python script to validate the data, ensuring accuracy by testing each file against specific criteria. After validating the data and securing it on an internal server, I presented the outcomes and tools to the stakeholders, demonstrating the project’s alignment with the initial goals and its successful execution.
